❓ Häufig gestellte Fragen (FAQ)

Wie kann ich meine Daten exportieren?

Öffnen Sie ein Protokoll → Tippen Sie auf „Exportieren" → Wählen Sie PDF, JSON oder ZIP → Speichern oder per E-Mail versenden

Werden meine Daten in der Cloud gespeichert?

Nein! Alle Daten bleiben lokal auf Ihrem Gerät. Eine Cloud-Speicherung erfolgt nur, wenn Sie Daten selbst exportieren und hochladen.

Was passiert, wenn ich mein Handy wechsle?

Exportieren Sie Ihre Daten vor dem Wechsel als ZIP-Datei. Nach Installation auf dem neuen Gerät können Sie die Daten importieren.

Sind die Premium-Features ein Abo?

Nein, alle Käufe sind einmalig. Es gibt keine Abonnements oder wiederkehrende Zahlungen.

Kann ich eine Rückerstattung erhalten?

Nach Freischaltung von Premium-Features ist keine Rückerstattung möglich. Bitte testen Sie die kostenlose Version ausgiebig.
